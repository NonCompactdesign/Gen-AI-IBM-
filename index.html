🔐 How to Get Your OpenRouter API Key

  1. Go to openrouter.ai
  2. Click on “Sign In” with GitHub or Google
  3. After logging in, visit openrouter.ai/keys
  4. Click “Create new key” and copy the key that starts with sk-
  5. Paste this key into the app when prompted
